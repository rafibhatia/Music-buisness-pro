# Music Business Pro - Installation Guide

## Overview

Music Business Pro is a Progressive Web App (PWA) that can be installed on your computer and run as a standalone desktop application. This guide will walk you through the installation process.

## What is a PWA?

A Progressive Web App is a web application that can be installed on your computer and run like a native desktop application. Benefits include:

- **No Browser UI**: Runs in its own window without browser tabs and toolbars
- **Quick Launch**: Access directly from your desktop or taskbar
- **Offline Access**: Continue working even without an internet connection
- **Automatic Updates**: Always get the latest features without manual updates
- **Cross-Platform**: Works on Windows, Mac, and Linux

## Installation Instructions

### Windows (Chrome/Edge)

1. Open Music Business Pro in Chrome or Microsoft Edge browser
2. Look for the **install icon** (computer with download arrow) in the address bar on the right side
3. Click the install icon, OR click the **three-dot menu (⋮)** and select "Install Music Business Pro..."
4. Click "Install" in the popup dialog
5. The app will open in its own window and be added to:
   - Your Desktop (if you checked that option)
   - Your Start Menu
   - Can be pinned to your Taskbar (right-click the app and select "Pin to taskbar")

**Alternative**: If you don't see the install icon, look for a notification banner with an "Install" button at the top or bottom of the page.

### Mac (Chrome)

1. Open Music Business Pro in Chrome browser
2. Look for the **install icon** (computer with download arrow) in the address bar
3. Click the install icon, OR click the **three-dot menu (⋮)** and select "Install Music Business Pro..."
4. Click "Install" in the popup dialog
5. The app will open in its own window and be added to your Applications folder. You can:
   - Find it in Applications
   - Add it to your Dock (drag from Applications to Dock)
   - Open it with Spotlight (⌘ + Space, then type "Music Business Pro")

### Mac (Safari)

1. Open Music Business Pro in Safari
2. Click the **Share button** in the Safari toolbar (square with arrow pointing up)
3. Select **"Add to Dock"** from the menu
4. The app icon will appear in your Dock. Click it to open the app in standalone mode

## Troubleshooting

### Don't see the install option?

- Make sure you're using a supported browser (Chrome, Edge, etc.)
- Try refreshing the page
- Check that you're accessing the app via HTTPS (secure connection)
- Look for a notification banner at the top or bottom of the page

### Already installed but can't find it?

**Windows:**
- Check Start Menu
- Check Desktop
- Search for "Music Business Pro"
- Access through browser's app menu: chrome://apps in Chrome

**Mac:**
- Check Applications folder
- Use Spotlight search (⌘ + Space)
- Check your Dock

### Want to uninstall?

**Windows:**
- Right-click the app and select "Uninstall"

**Mac:**
- Move the app from Applications to Trash

**Alternative:**
- Use your browser's settings to manage installed web apps
- Chrome: Settings → Apps → Manage apps

## In-App Installation Guide

Music Business Pro includes a built-in installation guide that you can access:

1. When you first visit the app, you'll see a welcome screen with installation instructions
2. From the app's Settings page, go to the "Appearance" tab and click "View Installation Guide"
3. Or navigate directly to: `?view=guide`

## System Requirements

- **Operating System**: Windows 10+, macOS 10.15+, or Linux
- **Browser**: Chrome 90+, Edge 90+, Safari 14+, or Firefox 90+
- **Internet**: Required for installation and initial setup; works offline after installation
- **Disk Space**: Approximately 50MB

## Features Available Offline

Once installed, the following features work offline:
- View existing contacts
- Browse communications history
- Access project information
- View analytics (cached data)
- Basic app navigation

**Note:** Real-time communications (SMS, calls, emails) require an internet connection and backend integration with services like Twilio and SendGrid.

## Security & Privacy

- Music Business Pro runs entirely in your browser with standard web security
- For full functionality with real communications, backend integration is required
- This app is designed for prototyping and development purposes
- Not intended for collecting PII or securing highly sensitive production data

## Support

If you encounter any issues during installation:

1. Check that you're using a supported browser and version
2. Clear your browser cache and try again
3. Ensure you have a stable internet connection during installation
4. Try using a different supported browser
5. Consult the in-app installation guide for detailed troubleshooting

## Additional Notes

### Desktop App vs Browser
- The installed desktop app and browser version share the same data
- Changes made in one will reflect in the other
- You can use both simultaneously

### Updates
- The app automatically checks for updates when you open it
- Updates are applied in the background
- No manual update process required

### Multiple Installations
- You can install the app on multiple computers
- Each installation can sync data if backend integration is configured
- Data is stored locally on each device
